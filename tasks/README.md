# Tasks Directory

The canonical backlog lives in GitHub Issues and GitHub Project.

Use this directory only for temporary local notes, rough drafts, or scratch files created while executing one lifecycle stage.

Rules:

- Final initiative and task definitions live in GitHub Issues.
- Final stage status lives in GitHub Project.
- Final analysis, delivery handoff, rollout, and validation decisions live in `docs/`.
- Do not treat files in `tasks/` as the durable backlog or the canonical specification.
